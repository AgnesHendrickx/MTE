Case study: Mount Etna

Additionally, two other optional modules are available exclusively for the Mount Etna case study, where the field sites from [MdG23] are reproduced:

  1. set_measurement_parameters.py(): This module provides DEM, site and path-specfiic values suach as the file to read, the location and number of measurement points, IGRF (International Geomagnetic Reference Field) values for the site, and more. See Functions for more details.

  2. etna.py: This module you define input parameters (see Case study: Mount Etna), to run different options reproducing field paths, to call set_measurement_parameters.py()

Finally, there are two scripts, script_etna_full.sh and script_etna.sh, that automatically modify etna.py, see Case study: Mount Etna for more information.

To run all (optimized) setups for case study Mount Etna (reproducing) field sites use script_etna_full.
./script_etna_full

Parameters of Etnean lava flows

For both the flank simulations and the final reproduction of the field sites of [MdG23] several values were chosen aiming to best simulate Etnean lava flows. What follows is the rationale for these choises.
The uniform remnant magnetization intensity assigned to the matter was: \(7.5\) \([A/m]\). This value compares well with both the TRM found in samples taken from the lava flows in the field [MdG23] and with previous paleomagnetic studies of Etnean lavas [NCB+14]; a bulk magnetization intensity of \(8\) \([A/m]\) was measured, with values ranging between \(5\) and \(13\) \([A/m]\). However, both much lower and higher values have also been reported in other studies, \(0.1-1\) \([A/m]\) and \(20\) \([A/m]\) by [SBC+06, TLG04], respectively. Clearly displaying the large dispersion of the measurements or values of the magnetization in Etnean lavas. The inclination of the magnetization used in this study is \(57 ^{\circ}\). This value was computed from Mount Etna’s average latitude using the Geocentric Axial Dipole hypothesis, (\(\tan{I} = 2\tan({lat})\)). The declination was assumed to be \(0 ^{\circ}\) (parallel to the present geomagnetic field) and from this the components of the magnetization were calculated.

When a flat bottom is chosen for the flank simulations, the input value for depth of the domain is added in the lowest value from the DEM for the area chosen, to avoid intersection situations as outlined in the note in flank simulations, however, this does if one increases domain depth, it depends on the size of the DEM cut chosen (and thus included features), how much depth is added underneath the path.
